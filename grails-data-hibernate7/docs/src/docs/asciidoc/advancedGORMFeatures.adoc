== Advanced GORM Features

This section covers advanced GORM and Hibernate mapping capabilities available through the `static mapping {}` DSL and other configuration mechanisms.

The ORM DSL `mapping` block is available on every domain class and allows you to customise every aspect of the Hibernate mapping:

[source,groovy]
----
class Book {
    String title
    Date dateCreated

    static mapping = {
        table     'books'
        title     column: 'book_title', index: true
        batchSize 20
        cache     usage: 'read-write'
    }
}
----
